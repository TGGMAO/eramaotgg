勇者のおさいふ事情改善パッチ

以前のカルマパッチの影響で勇者がすぐに借金まみれとなり
第1階層から降りてこなくなる問題が発生していたのをすこしだけ改善
勇者の収入にダンジョンの難度だけでなく勇者自身のレベルが影響するようになった
これにより第1階層〜第5階層くらいではあまり借金が増えなくなった
さらに商人トラップの売上に階層が関係するようにした

商人トラップの下方修正的な改造ではなく
勇者が資金を集めやすく＋深層の商人は値段をふっかけてくる　なので
特に序盤の展開が苦しくなるようなことは無い（はず）

不幸の指輪とシュート、深層の商人の組み合わせは以前にも増して凶悪

リリースのタイミングの問題で旧バージョン準拠になってしまったのでそこだけ注意
最新版でも適用可能かもしれませんが未検証




具体的な改造内容

@GET_JUNK_ITEM
勇者側のダンジョン収入の基礎値で勇者レベルを考慮にいれるように
また、最終結果には現在の階層が影響するように

@SHOP_TRAP
商人トラップは深層ほど急激にぼったくってくるように



作者はカルマパッチと同じ
バグ直してくれた方、ありがとうございます

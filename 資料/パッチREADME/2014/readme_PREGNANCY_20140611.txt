■仕様
　・妊娠時・臨月時でも出撃可能のコンフィグを追加。
　・迎撃中に出産可能。
　・妊娠時に捕獲されてNTR先で出産有り。
　・NTR先で狂王、VA乱交、公衆便所、獣姦ショーで妊娠＆出産を追加。
　・各口上にNTR出産口上追加、テンプレ追加。
　・妊娠出産機能オフ時でも中だしフラグをリセットします。

　■eramaouフラグまとめ.txt
　CFLAG:5に各種フラグ追加。
　CFLAG:102 = 誰によって妊娠させられたか（マスター = 1, 助手 = 2, 奴隷 = 3, 客 = 4, 犬 = 5 モンスター・触手 = 6 狂王 = 7）
　CFLAG:108 = 狂王からの中田氏カウント用

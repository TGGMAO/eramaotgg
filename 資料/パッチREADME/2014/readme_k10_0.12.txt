eramaou_k10_ver0.12

このファイルはeramaou用「クラブ」口上に加筆修正したものです

※(eramaou　ver.0.290+eramaou_0290パッチまとめ20140414.zip適用を推奨)

加筆・改変・再アップロードはご自由にどうぞ。

キャラコンセプトは
メガネロリアラサーひきこもりふたなり魔法使い。あなたにどこか馴れ馴れしいキャラ。

■使い方
EVENT_K10.ERBファイルをERBフォルダに上書きコピーしてください。

■攻略法…というか注意
　まだ調教が進んでない段階でふたなり射精すると一発で反発刻印Lv３がついてしまって屈服の珠を溜めるのに恐ろしく手間が掛かってしまうので
　いっそのこと実験室でふたなりをとってしまいしょう。
　その場合は口上で矛盾する部分が出て来ますがすみません。
　
■仕様
ver0.12
・調教開始時の簡易助手口上に加筆。
・調教開始時にバニースーツ、メイド服、娼婦のドレスの口上追加。

ver0.11
・NTR自慰口上追加、NTRれて奪還したあと調教後自慰でＮＴＲ自慰口上が見れます。
・調教後自慰に自慰狂い追加。
・調教開始時と終了時の反発刻印Lv3口上を修正。
・反発刻印Lv3取得時の口上を修正。
・一部口上（自慰）を修正。

ver0.10
簡易助手口上はアルカナナイトのみに対応。
獣姦とクリキャップ以外のテンプレにある全ての口上に加筆。
フェラする、逆レイプ、逆アナルレイプに加筆。
【自慰狂い】に対応。

■使い方
　全てのERBファイルをERBフォルダに上書きコピーしてください。

■仕様
・メインメニューのアイテム/スキル一覧で【魔蟲知識】を表示。
・素質分類表示で人妻を表示。
・アイテム購入時に技巧レベルと調教アイテムと罠レベルと罠の一覧を表示。
・【淫魔知識】【魔蟲知識】を何度も購入出来る不具合を修正。

■追加・修正箇所
　■_DRAW_MAINMENU.ERB
　　302行目
　	　SIF TALENT:MASTER:56 == 1
　　		PRINT 【魔蟲知識】

　■CHARA_INFO_SHOW.ERB
　　256行目
　　;157,人妻,;結婚した証。欲望が上がりやすい
　　	SIF TALENT:TARGET:157
　　		CALL SHOW_TALENT_GROUP(157)

　■SHOP_ITEM.ERB
　　34〜36行目
　　購入した調教アイテムと消費型調教アイテム一覧を表示するように追記。
　　175行目
	ITEMSALES:54 = 0
　　183行目
	ITEMSALES:56 = 0

　■SHOP_TRAP.ERB
　　23〜53行目
　　購入した罠一覧と指輪一覧を表示するように追記。

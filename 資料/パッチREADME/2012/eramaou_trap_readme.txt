eramaou_trap

eramaou用「DUNGEON_ROOM.ERB」「DUNGEON_TRAP.ERB」「SUMMON_MONSTER.ERB」に加筆修正したものです。

※(eramaou　ver.0.210推奨)

■使い方
DUNGEON_ROOM.ERBとDUNGEON_TRAP.ERBとSUMMON_MONSTER.ERBを上書きしてください。

■仕様
DUNGEON_ROOM.ERB
いまいち効果がわかりにくかったダンジョン施設の効果に加筆しました。
DUNGEON_TRAP.ERB
罠に引っかかっても何ポイントのダメージを与えたか分からないものがあったので加筆しました。
SUMMON_MONSTER.ERB
召喚の罠が発動したときに誤作動を起こさないようにしました。

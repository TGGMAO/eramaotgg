eramaou_hobbit_dwarf_mazoku_ver0.01

※(eramaou　ver.0.145推奨)

■使い方
LOOK.ERBファイルとEVENT_NEXTDAY.ERBファイルとCHAR_MAKE.ERBファイルをERBフォルダに上書きコピーしてください。

■仕様
午後の行動を終えた後
【青肌】【悪魔羽】【悪魔尻尾】【悪魔眼】が全て揃っているキャラクターは種族が魔族になります。
選択無しの強制ですのでお気をつけください。


・種族の追加
種族：ドワーフ
種族特性として【小柄体型】【貧乳】になります
ドワーフは陰毛が剛毛になりやすいです

種族：ホビット
種族特性として【小柄体型】【貧乳】になります
ホビットは陰毛が無毛になりやすいです

※(eramaou　ver.0.270推奨)

加筆・改変・再アップロードはご自由にどうぞ。

この圧縮ファイルはSELL_MATURO.ERBを含む売却時の拡張口上詰め合わせです。

■使い方
　全てのERBファイルを上書きしてお使いください

■仕様
2013/09/06
・末路口上の誤字脱字修正と分岐条件の修正。

・愛と淫乱の陥落条件に追加条件。
　レズ条件も無くなったし、さすがにちょろすぎるので条件に屈服刻印その他ぐらいはつけてもバチはあたらないと思うの。

・売却時に素質による倍率表示が見えるようにしました。

・売却時の倍率の提案。
　売却値の修正が常に６倍も（TALENT:310、陰毛の状態とTALENT:311、陰毛生育限界は常にフラグが立っているので）入るのは思案所でしたので
　TALENT:165村娘以外のキャラ（つまり元勇者）に３倍の修正にして減らすようにしました。
　それに伴いTALENT:パイパン、筋肉質、褐色肌、白皙をもっていた場合に売却値にプラス修正がかかります。

　これは後に狂王等の身分の高いキャラが追加された場合の差別化です。

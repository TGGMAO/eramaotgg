eramaou_jisinka_ver0.10

このファイルはeramaou用「自信家」口上に加筆修正したものです

※(eramaou　ver.0.280推奨)

加筆・改変はご自由にどうぞ。

■使い方
EVENT_K1.ERBファイルをERBフォルダに上書きコピーしてください。

■仕様
ver0.10
処刑口上を追加。
末尾に@EXUCUTION_KOUJO_K1関数を追加。

末尾に@NTR_KOUJO_K1関数を追加。
調教開始時台詞にNTR再捕獲を追加。
胸愛撫に母乳体質分岐を追加、ニプルキャップと搾乳機を装着していなければ分岐します。
パイズリ時のフラグを修正（CFLAG:243からCFLAG:333に）

ver0.09
全体的な誤字脱字の修正、見直し。
一部に崩壊口上の追加。
一部に助手口上の追加。
「ピアシング」口上を加筆。
「何もしない」口上を加筆

ver0.08
「簡易助手口上」を追加、村娘が助手の時に流れます。

ver0.07
「会話する」に加筆
「調教後自慰」にレズ自慰と野良犬自慰を加筆

ver0.06
調教前魔族化、陥落前魔族化、魔族陥落時の口上追加

ver0.05
助手調教時に正常位、後背位、対面座位、背面座位、騎乗位
正常位アナル、後背位アナル、対面座位アナル、背面座位アナル、騎乗位アナル時の口上を追加しました

パラメータ変動をトリガーにした口上
刻印変動をトリガーにした口上
において
ボールギャグ着用時、失神時、獣姦プレイ中、触手調教中は口上スキップ。

ver0.04
初調教時に種族別に口上が出るように追加
助手、野良犬、触手調教時に口上スキップ

ver0.03
判明してるバグ部分を修正

ver0.02
試験的に一部にカラー設定追加
口上追加、及び一部修正
	【淫乱】時の口上を追加
	【淫乱】+【アナル狂い】時の口上追加

種族が魔族の時に以下の口上を追加
	陥落初回時、調教開始時、調教終了時の台詞
	【愛】【淫乱】時に処女正常位、処女後背位、処女騎乗位の口上

ver0.01
元のファイルへの口上の追加、及び一部修正

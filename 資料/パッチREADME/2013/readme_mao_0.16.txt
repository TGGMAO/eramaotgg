eramaou_mao_ver0.15

このファイルはeramaou用「村娘A」口上に加筆修正したものです

※(eramaou　ver.0.280推奨)

加筆・改変・再アップロードはご自由にどうぞ。

キャラコンセプトは
生意気ロリキャラの陥落、とらわれのお姫様（仮）、助手調教時の口上追加キャラ

ゲーム開始時にランダム生成勇者と村娘の二択。

■使い方
EVENT_K5.ERBファイルをERBフォルダに上書きコピーしてください。

■仕様
ver0.16
処刑口上を追加。
末尾に@EXUCUTION_KOUJO_K5関数を追加。

ver0.15
NTR口上追加
末尾に@NTR_KOUJO_K5関数を追加。
調教開始時台詞にNTR再捕獲を追加。
パイズリ時のフラグを修正（CFLAG:243からCFLAG:333に）
胸愛撫に母乳体質分岐を追加、ニプルキャップと搾乳機を装着していなければ分岐します。
調教開始時に裸時の口上を追加、普段着（ズボンorスカート）、メイド服、娼婦のドレス着用時の口上追加。

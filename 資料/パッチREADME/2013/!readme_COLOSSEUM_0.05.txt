;戦闘値の計算
を大部分変更
レベルに比例＆上位のモンスターになると調教対象に対して一方的に攻撃出来るようになります
仕様的に調教対象の気力を０にしてしまえば＜奴隷陥落＞するので関係ないのですが…

最初に減らされる体力を少なくするように修正、気力が０の場合はさらに1/4に修正されます。

気力があれば追加ダメージがありますが
調教対象の気力が０の場合追加ダメージはありません

モンスターの攻撃で次のターンの気力が０になる場合＜奴隷陥落＞とメッセージが出ます

フェラチオ実行が不可時にRETURN 0で返るように修正

後輩位で処女を奪わせない時にRETURN 0で返るように修正

各種コマンドから別コマンドへのジャンプを統合…ってなんで以前はわざわざ二つに分けるようなことを…

コロシアム収入を増加
調教対象を＜奴隷陥落＞にさせてエッチなことをすると収入有り、薬代ぐらいにはなるはず

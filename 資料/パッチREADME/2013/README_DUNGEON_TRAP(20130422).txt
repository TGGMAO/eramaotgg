eramaou ダンジョン罠パッチ(2013/04/22)

※(eramaou　ver.0.260推奨)

■使い方
　ERBファイルを上書きしてお使いください

■対処した問題
階層に罠が設定されてなく振動の宝石を入手してない状態で罠自動補充をオンにすると「罠がない！」と表示される度に
ITEM:99（ITEM.CVSでは設定されていません）が99個になるまで自動的に購入し続けてお金が減っていく。

■上記の問題の原因
罠が階層に設置されてない場合ITEM:0があるかどうかチェックされ、ITEM:0が1個未満の場合は罠自動補充でITEM:99を購入することになるため。

■変更点
51-52行目の
		PRINTL 罠がない！
		Y = 99
を削除

97行目の
		IF MONEY >= RESULT && ITEM:Y < 99
を
		IF MONEY >= RESULT && ITEM:Y < 99 && Y >=60 && Y <= 78
に変更

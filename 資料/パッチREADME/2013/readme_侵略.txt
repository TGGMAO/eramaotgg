加筆・改変・再アップロードはご自由にどうぞ。

■仕様
2013/10/09
■INVASION.ERB
掲示板で指摘されたので文字変数Yを#DIM変数"SINKOU"に変更
それに伴いモンスター戦力のバランス調整
・魔王様の魔法を威力二倍にして経験値が入手出来るようにしました。
・精鋭部隊が勇者基礎レベル補正で強化されるようにしました。さらに攻撃力を強化しました。
・メッセージ表記を調整

■SYSTEM.ERB
486行目のCALL INVASION_CHECKをコメントアウト
470〜506行目に
侵略先の侵略度ダウンを明記
エルフの領域、ドラゴンの山、天界も侵略度ダウンするように設定。
午前午後で反抗します。

2013/08/30
■INVASION.ERB
勇者で侵攻時に敗北か引き分けをすると時間が進まなかったバグを修正。
298、371、406行目をRETURN 1に。

2013/08/15
■INVASION.ERB
勇者で侵攻した時、狂王の精鋭部隊と戦う場合があります。
NTRフラグオン時に敗北すると捕獲されてNTR状態になります。

■SHOP.ERB
345行目
ELSEIF CFLAG:COUNT:1 == 9
	SETCOLOR 255,0,0
	PRINT [NTR中] 
	RESETCOLOR
ELSE
メインメニューで能力の表示を選択時にキャラがNTRれているのならば[NTR中]と表示されるようにしました。

2013/04/25
「魔王の魔法を使う」に口上を追加しました。

2013/04/24
侵攻で勇者と共にモンスターの1/3が侵攻出来るようになります。
勇者を侵攻させると経験値を入手できます。
侵略選択時にエルフの領域、ドラゴンの山、天界の侵攻度を表示するようにしました。
勇者補正、魔王補正の表示の修正。

2013/04/21
侵攻で勇者を選べます。
愛か淫乱で助手が可能が条件です。
